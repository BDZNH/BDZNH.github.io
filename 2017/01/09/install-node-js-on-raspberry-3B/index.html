环境描述

  1. 树莓派安装了2016-11-25-raspbian-jessie-lite(PS：在此版本的镜像中，默认禁用了ssh，在烧录好镜像之后，在boot分区中新建一个名为ssh的文件夹，正常开机，通过ssh即可连接到树莓派)
  2. 所有官方软件源软件已经更新至最新版。
