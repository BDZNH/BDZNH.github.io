对排序的一点扯淡

发表于 2017-04-03 更新于 2024-07-27 阅读次数：
本文字数： 557 阅读时长 ≈ 2 分钟

对数据进行排序，是需要经常进行的操作。
手动写排序，归并，冒泡，插入之类的都可以，不过有algorithm自带的sort就省了一些麻烦了。
对需要排序的数组这样写sort(a,a+n)(假设a为需要排序的数组，n为数组的长度)
再细化一点的话，就这样写:sort(a+n1,a+n2)，这样是对a数组的下标为n1的元素直到下标为n2的元素进行排序，自然，n2>n1。
