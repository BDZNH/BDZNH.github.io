AOSP 使用 repo 来组织代码的，有时候我们只需要下载其中一部分代码。下面是具体的操作方式

比如我们要下载 apksig 的源码，就可以找到

把这里的 name 取出来，拼上https://mirrors.tuna.tsinghua.edu.cn/git/AOSP/得到一个链接https://mirrors.tuna.tsinghua.edu.cn/git/AOSP/platform/tools/apksig，用git clone https://mirrors.tuna.tsinghua.edu.cn/git/AOSP/platform/tools/apksig的形式就可以下载到对应的代码了。
